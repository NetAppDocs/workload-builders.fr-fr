=== Prise en charge du clonage

Vous pouvez désormais cloner un projet dans BlueXP Workload Factory for Builders.  Lorsque vous clonez un projet, Builders crée un nouveau projet à partir d'un instantané, avec la même configuration que l'original.  Le clonage est utile pour créer rapidement des projets similaires ou à des fins de test.  Vous pouvez monter le nouveau clone du projet en suivant les instructions dans Builders.

https://docs.netapp.com/us-en/workload-builders/version-projects.html["Gérer les versions des projets BlueXP Workload Factory for Builders"]
