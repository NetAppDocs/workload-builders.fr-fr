= Automatisez les tâches de Builders avec Codebox
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Vous pouvez automatiser les opérations de création de projets et de protection des données avec Codebox.  Codebox est un copilote d'infrastructure en tant que code (IaC) qui vous aide à générer du code pour exécuter toutes les opérations prises en charge par Workload Factory.

En savoir plus sur link:https://docs.netapp.com/us-en/workload-setup-admin/codebox-automation.html["Automatisation de la Codebox"^] et comment l'utiliser.
